# Cybersecurity Awareness Chatbot 🇿🇦

A C# console application that educates South African citizens about cybersecurity threats including phishing, password safety, and suspicious links.

## Features
- 🎵 Voice greeting on startup
- 🎨 ASCII art cybersecurity header
- 👤 Personalised responses using user's name
- 💬 Answers questions about:
  - Password safety
  - Phishing emails
  - Safe browsing
  - Suspicious links
  - South African cyber threats
- ✅ Input validation for empty responses
- 🔄 CI/CD with GitHub Actions

## How to Run
1. Open solution in Visual Studio
2. Press F5 to run
3. Type 'quit' to exit

## Requirements
- .NET Framework 4.7.2+ or .NET 6/7/8
- Windows (for full audio support)
